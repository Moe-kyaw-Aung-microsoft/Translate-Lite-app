# Translate Lite

Translate Lite is a lightweight multilingual translation project supporting:
- Myanmar (Burmese)
- English
- Chinese Simplified
- Chinese Traditional
- Thai

Features:
- Text translation
- Speech-to-Text
- Text-to-Speech
- Conversation mode
- Translation history
- Settings
- GitHub Pages landing site
